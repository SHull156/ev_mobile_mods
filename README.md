# E. V. Mobile Mods – Electric Vehicle Specialist

A simple, fast, responsive website for a mechanic specialising in electric and hybrid vehicles.  

## Overview
This project is a brochure-style website designed to provide clear information about services, service areas, and contact details for a local business. The focus is on clarity, accessibility, performance, and mobile-first design.

## Features
- Responsive layout (mobile-first)
- Clear call-to-action (click-to-call)
- Services overview
- Contact section with map and opening hours
- Basic on-page SEO
- Fast static deployment

## Tech Stack
- HTML
- CSS
- JavaScript (minimal)
- Netlify (deployment)

## Getting Started
To run locally:
1. Clone the repository
2. Open index.html in your browser  
(No build step required)

## Deployment
The site is deployed using Netlify and connected to this GitHub repository for automatic deployments.
